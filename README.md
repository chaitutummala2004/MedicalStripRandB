# Smart Pharmacy Management System

An AI-powered Pharmacy Management System with prescription OCR, real-time medicine strip detection, and automated billing.

## Features
- **Prescription Scan**: Extract medicines from handwritten/printed prescriptions.
- **Strip Detection**: Real-time recognition of medicine strips via camera.
- **Inventory Management**: Track stock, prices, manufacturing, and expiry dates (FEFO).
- **Billing & Reports**: Generate receipts with customer details and view sales history.
- **Medicine Info**: Quick Google lookup for medicine usage and safety.
- **Performance Analysis**: Built-in evaluation for precision and recall metrics.

## Installation
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd USingTrae
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Access in browser: `http://localhost:5000`
